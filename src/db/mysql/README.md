## Assessment Manager Database

### Setup (run once to setup the database)
```
./amdb.sh setup
```

### Clear the database
```
./amdb.sh setup
```

### Connect to the database locally
```
./amdb.sh connect
```

### Connect to the database as root
```
./amdb.sh connect.admin
```

### Load test data
```
./amdb.sh load
```

### Clear and remove the database completely
```
./amdb.sh drop
```
